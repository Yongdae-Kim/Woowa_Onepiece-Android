# Android Client

***

## 원피스 프로젝트

### 제주 버스 정류장의 위치 정보를 중심으로 버스 시간 및 주변 볼거리, 먹거리, 광고 정보를 제공해 주는 서비스

***

## 개발환경

- Java version : 1.8
- Retrolamda version : 3.2.4
- Android Min Sdk Version : 16 (Android 4.1 Jelly Bean)
- Android Target Sdk Version : 23 (Android 6.0 Marshmallow)

***
